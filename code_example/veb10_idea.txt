Шпаргалалка идей.
Что уже сделано.

1. Простые функции - 3+ реализаций.
Не очень интересны, лучше спросить у жпт.
- upper(), lower(), title(), capitalize(), swapcase()

2. Известные шифры. Азубка Морзе - 2 реализации.
- Нужно учесть задержку звукового сигнала - передача побуквенная.
- латиница, кириллица, знаки, числа
- можно дополнительно: подсчитать скорость передачи сообщения

3. Известные шифры. Шифр Цезаря - 1 реализация, но тут широкий простор.
Простой вариант "безопасной" передачи данных.
Варианты реализации:
- через словарь с фиксированным смещением
- через обработку строки, с гибким смещением
    list('абвгдеёжзийклмнопрстуфхцчшщъыьэюя')
- через ord() + chr(), с гибким смещением.
- поддержка латиница, кириллица, регистра, знаки, числа
- отдельно можно сделать фильтр декодирования

4. Комбинирование встроенных функций и доработки. Полный простор фантазии.
- словари - для создания шифров с собственной логикой. Эмодзи и жпт - как источник идей. Можно использовать одну функцию для обработки разных словарей.
- поддержка латиница, кириллица, регистра, знаки, числа
- replace(" ", "_") - замена в строках - хороший метод для реализации.
- рандомные генераторы - полезны для твоческих идей

4.1. Worms filter - червяк
    Преобразовывает все согласные в "_", а гласные в "-", добавляем голову-эмодзи
4.2. Burr filter - картавость
    Преобразовывает букву "р" в "л", а букву "к" в "т".
4.3. lisp filter - шепелявость
    Преобразовывает звуки "с", "з" и "т" в звуки "ш", "ж" и "ч" для русского языка,
    звуки "s" и "z" в звук "th" для английского языка.
4.4. Random change с усложнением.
    Меняет каждую букву на рандомную, а пробел на рандомный символ.
    У пользователя есть дополнительный выбор: поменять буквы и пробелы, поменять только буквы, только пробелы.
4.5. Random color
    Окрашивает каждый символ текста в рандомный цвет.
